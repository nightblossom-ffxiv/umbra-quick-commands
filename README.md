# Umbra Quick Commands

A configurable Umbra toolbar widget that turns your toolbar into a launcher for your favourite Dalamud slash commands.

Add up to 16 entries (Label / Command / Icon) in the widget settings, then click any entry in the popup to fire the command — the same as typing it in chat. Works with anything Dalamud accepts: `/penumbra`, `/glamourer`, `/heels`, `/moodles`, `/ptodo`, `/wordsmith`, etc.

## Install

1. Download `UmbraQuickCommands.dll` from the [latest release](https://github.com/nightblossom-ffxiv/umbra-quick-commands/releases/latest).
2. In Umbra: Settings → Plugins → Install local plugin → pick the DLL.
3. Author: `Nightblossom Team`, Repository: `nightblossom-ffxiv/umbra-quick-commands`.
4. Add the **Quick Commands** widget to your toolbar.
5. Open its settings and fill in the Slot 01..16 entries with whatever commands you want.
